# Skill Catalog

Available skills for your AI team. Add your own as your workflow grows.

## Core Skills

| Skill | Command | What it does |
|-------|---------|-------------|
| **AI Team Architecture** | `/ai-team-architecture` | Design AI agent teams using OGSM + 3-layer architecture + multi-model routing |
| **OGSM Framework** | `/ogsm-framework` | Strategic planning with OGSM methodology (Objective, Goals, Strategies, Measures) |
| **AI Collab** | `/ai-collab` | Task-type-aware multi-model router (Claude / Gemini / Codex) |
| **AI Fallback** | `/ai-fallback` | Quota-aware, hang-resilient multi-model fallback chain |
| **Agent Builder** | `/agent-builder` | Create or modify AI agent roles in your team |

## Common Workflows

### Workflow A — Set up a new AI agent team
```
/ai-team-architecture → define OGSM → assign roles → configure routing → deploy
```

### Workflow B — Multi-model content production
```
/content-scout (find topics) → /ai-collab (route research to Gemini, writing to Claude) → /writing-guide (quality check)
```

### Workflow C — Add or modify an agent
```
Tell your team leader what you need → it asks clarifying questions → builds the role
```

## You Don't Need to Type Commands

Your AI team leader detects your intent and suggests the right skill automatically. Just describe what you want to do in plain language.
